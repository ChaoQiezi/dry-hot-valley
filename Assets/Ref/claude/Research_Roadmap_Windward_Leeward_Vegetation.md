#### Figure 6: GAI/GDI 空间分布图（Core Asymmetry Map）

**这是整篇论文最核心的图，所有参考论文都有。**

**内容与布局**：

- **(a)** 多年平均GAI（或GDI）空间分布图（3×3 km格网）
  - 色标：GAI < 1（背风坡更绿）→ 棕橙色；GAI > 1（迎风坡更绿）→ 深绿色
  - 发散色标，中心值=1（白色）
- **(b)** 显著性检验结果图（t-test, p值分三级：p<0.01, 0.01<p<0.05, p>0.05）
- **(c)** 可选：按海拔段统计的GAI分布直方图或箱线图

**计算方法**：

```
对每个3×3 km格网：
1. 提取所有迎风坡像元的7年峰值NDVI，取均值 → NDVI_windward
2. 提取所有背风坡像元的7年峰值NDVI，取均值 → NDVI_leeward
3. GAI = NDVI_windward / NDVI_leeward
4. 两样本t-test检验差异显著性
过滤条件：
- 坡度 5°-25°（排除平地和陡坡的地形效应）
- NDVI > 0.1（排除无植被像元）
- 格网内迎/背风坡像元数均 ≥ 50
```

**科学目的**：揭示迎/背风坡植被不对称性的空间异质性——哪些区域迎风坡更绿，哪些区域背风坡更绿。

**参考**：Yin 2023a Figure 1（GDI全球图）；Xie 2025 Figure 2a（GAI空间图）。

#### 关键数据预处理：坡度过滤 + 土地覆被过滤

**在计算任何GAI/GDI之前，必须先做以下过滤：**

1. **坡度过滤**：仅保留坡度在 5°-25° 之间的像元
   - 理由：<5° 基本为平地，Wind Effect分类不稳定；>25° 受地形阴影影响大，NDVI存在系统偏差（Chen et al., 2020）
   - 所有参考论文都做了此过滤

2. **土地覆被过滤**：至少排除以下地类：
   - 水体、冰川/永久积雪、裸地/裸岩
   - 建议保留：森林、草地、灌木，可分别做分层分析
   - 推荐使用 ESA WorldCover 10m（与Sentinel-2分辨率匹配）或 MCD12Q1

3. **NDVI阈值**：保留 NDVI > 0.1 的像元

**这三步过滤极其重要，因为你在LOG中提到的"迎风坡整体NDVI低于背风坡"的反直觉现象，很可能就是因为高海拔裸地/冰雪像元被纳入了迎风坡统计。**

## 四、你的研究新颖性分析

### 与参考论文的关键差异

| 维度 | 参考论文 | 你的研究 |
|------|----------|----------|
| **不对称性驱动力** | 辐射驱动（EFS vs PFS），坡向角简单划分 | **水分/风驱动**（迎风 vs 背风），基于物理的Wind Effect方法 |
| **分类方法** | 坡向角范围（如135°-225°为南坡） | SAGA-GIS Wind Effect指数 + ERA5风场数据 |
| **地理区域** | 青藏高原或全球山区 | **川西干热河谷区**——全球著名的干热河谷集中区 |
| **海拔梯度** | 仅格网化分析，未系统分海拔段 | **海拔梯度精细分析**（50m间隔），发现4100m反转 |
| **空间分辨率** | MODIS 250m / Landsat 30m | **Sentinel-2 10m**（当前最高的免费光学卫星分辨率） |
| **干热河谷** | 未涉及 | **核心关注**——焦效应(foehn effect)、干热河谷植被退化 |

### 核心卖点（Selling Points）

**卖点1：从辐射驱动到水分驱动的范式转换**

参考论文的核心逻辑是：坡向 → 辐射差异 → 温度/水分微气候差异 → 植被差异。你的研究跳过了辐射中介，直接从**风场**入手：风向 → 迎/背风效应 → 降水截获/焦效应 → 水分差异 → 植被差异。这是一个从"被动接收辐射"到"主动拦截水汽"的机制转换。

**卖点2：海拔依赖的反转现象**

你的数据清晰地展示了 ~4100m 处的GAI反转：低海拔迎风坡更绿（水分主导），高海拔背风坡更绿（温度主导/风寒效应）。这一发现在参考论文中**从未被报道**——因为它们的PFS/EFS框架无法捕捉到这种"谁拦截了更多水汽"的维度。

**卖点3：干热河谷——全球意义的区域案例**

川西干热河谷是全球范围内最典型的河谷型干旱环境之一，是焦效应（foehn effect）的教科书案例。在这些河谷中，迎风坡与背风坡的植被差异不仅仅是统计量上的不同，而是直接关系到生态恢复和防灾减灾的实践问题。这赋予你的研究直接的应用价值。

**卖点4：10m分辨率——前所未有的精细地形-植被匹配**

你使用Sentinel-2 10m分辨率，远超MODIS 250m和Landsat 30m，能够更精确地区分不同坡向的微地形单元。这降低了混合像元效应对GAI/GDI的干扰。

#### Abstract (~200 words)
- 背景：坡向通过微气候调控植被生长已被广泛认知，但以往研究聚焦于辐射驱动的PFS/EFS差异，wind-driven的迎/背风坡植被不对称性尚未被量化
- 方法：利用Sentinel-2 10m NDVI（2019-2025）+ Wind Effect指数 + ERA5风场，在3×3 km格网内计算GAI

- 结论：风驱动的植被不对称性是山地植被空间格局的重要但被忽视的维度

  - 与PFS/EFS框架的对比——辐射 vs 水汽
- **4.2** 海拔依赖的反转——从水分限制到温度限制
  - 低海拔（干热河谷）：水分为主要限制因子 → 迎风坡水分更丰富 → GAI > 1
  - 高海拔（>4100m）：温度为主要限制因子 → 迎风坡风寒效应更强 → GAI < 1
  - 与Yin 2023a的全球GDI-MAT-P/PET结论呼应
- **4.3** 干热河谷的生态意义
- **4.4** 局限性
  - 7年时间序列短，趋势分析受限
  - 600hPa风场可能不完全代表地表风
  - Wind Effect二分类的阈值简单性
  - 无地面验证
- **4.5** 未来研究方向
